How many IPv6 adresses exist?

June 1, 2012
Some general information A typical IPv4 adress in a lokal network looks like this: 192.168.0.1 This is 11000000 10101000 00000000 00000001 in binary octets. You can easily see that it has 32 bits. As one bit may have two values - 0 or 1 - there are $2^{32} = 4,294,967,296 \approx 4.3 \cdot 109$ possible addresses.
