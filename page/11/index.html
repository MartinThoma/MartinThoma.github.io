Add a new font to ImageMagick

March 6, 2013
You can list all fonts that are known to ImageMagick by identify -list font. When your font isn't there, but it is installed, you might want to try these steps: sudo updatedb Download imagick_type_gen Execute it: perl imagick_type_gen > types.xml Copy the result to the folder where it should be locate type.xml. That was /usr/lib/ImageMagick-6.5.7/config/type.xml for me sudo cp type.xml /usr/lib/ImageMagick-6.5.7/config/type.xml You can find some nice fonts here.
