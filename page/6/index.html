The Rust Programming Language

Rust is a compiled programming language which aims to replace C++. The designers wanted it to be similar fast, give the programmer similar fine-grained control over memory management, but more safety. Rust does so by introducing the concept of ownership, borrowing and lifetimes of variables. Rust also makes sure that after the scope is finished, the memory of variables gets de-allocated.
