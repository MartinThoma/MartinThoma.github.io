Function Approximation

January 18th, 2016

I was recently quite disappointed by how bad neural networks are for function
approximation (see How should a neural network for unbound function approximation be structured?). However, I’ve just found that
Gaussian processes are great for function approximation!

There are two important types of function approximation:

  • Interpolation: What values does the function have in between of known
    values?
  • Extrapolation: What values does the function have outsive of the known
    values?

I did a couple of very quick examples which look promising.

Examples

Square

Approximating \(f(x) = x^2\) worked very good:

f(x) = x^2

I’ve tried if with higher order polynomials, more complex polynomials. No
problem.

Sin

Approximating \(f(x) = \sin(3x)\) seems to be more complicated:

f(x) = sin(3x)

I guess a human would see the wave pattern and do a better job here.

exp

Approximating \(f(x) = e^x\) works similar well as polynomials. One can see
that it does not perfectly fit it, but compared the the range of values seen
before and the distance from the last seen value I think this is absolutely
acceptable:

f(x) = e^x

noise

It is claimed that Gaussian processes implicitly model noise so that they can
easily deal with noise. However, in my experients this seems not to work so
great. The reason might be that I had points in \([-3, 3]\) of the function

\[f(x) = x^2\]

with point-wise gaussian noise \(N \sim \mathcal{N}(0, 1)\). So the noise is
quite domintant on that intervall. One of the examples where it worked better
is:

f(x) = x^2 with gaussian noise

Make it brake

I was a bit suspicious if I had another mistake here. So I wanted it to break.
This was the reason why I created the following function

\[f(x) = \begin{cases}x^2 &\text{if } x \geq 0\\-1 &\text{otherwise}\end{cases}\]

Function with discontinuity

The predicted value is obviously not correct, but you should note that almost
all function values are within the 95% confidence intervall!

Code

The following code needs numpy
and sklearn. For the plots,
you need matplotlib.

 1#!/usr/bin/env python
 2
 3"""Example how to use gaussion processes for regression."""
 4
 5import numpy as np
 6from sklearn import gaussian_process
 7
 8
 9def main():
10    # Create the dataset
11    x_train = np.atleast_2d(np.linspace(-3, 3, num=50)).T
12    y_train = f(x_train).ravel()
13    x_test = np.atleast_2d(np.linspace(-5, 5, 1000)).T
14
15    # Define the Regression Modell and fit it
16    gp = gaussian_process.GaussianProcess(theta0=1e-2,
17                                          thetaL=1e-4,
18                                          thetaU=1e-3)
19    gp.fit(x_train, y_train)
20
21    # Evaluate the result
22    y_pred, mse = gp.predict(x_test, eval_MSE=True)
23    print("MSE: %0.4f" % sum(mse))
24    print("max MSE: %0.4f" % max(mse))
25    plot_graph(x_test, x_train, y_pred, mse, "x^2")
26
27
28def f(x):
29    """
30    Function which gets approximated
31    """
32
33    noise = [np.random.normal(loc=0.0, scale=1.0) for _ in range(len(list(x)))]
34    noise = np.atleast_2d(noise).T
35    return x**2 + noise
36    # Totally fails for that one:

39    #     if el >= 0:
40    #         y.append(el**2)
41    #     else:
42    #         y.append(-1)
43    # return np.array(y)

45
46def plot_graph(x, x_train, y_pred, mse, function_tex):
47    # Plot the function, the prediction and the 95% confidence interval
48    # based on the MSE
49    sigma = np.sqrt(mse)
50    from matplotlib import pyplot as pl
51    pl.figure()
52    y = f(x_train).ravel()
53    pl.plot(x, f(x), 'r:', label=u'$f(x) = %s$' % function_tex)
54    pl.plot(x_train, y, 'r.', markersize=10, label=u'Observations')
55    pl.plot(x, y_pred, 'b-', label=u'Prediction')
56    pl.fill(np.concatenate([x, x[::-1]]),
57            np.concatenate([y_pred - 1.9600 * sigma,
58                           (y_pred + 1.9600 * sigma)[::-1]]),
59            alpha=.5, fc='b', ec='None', label='95% confidence interval')
60    pl.xlabel('$x$')
61    pl.ylabel('$f(x)$')
62    y_min = min(min(y_pred), min(y)) * 1.1
63    y_max = max(max(y_pred), max(y)) * 1.1
64    pl.ylim(y_min, y_max)
65    pl.legend(loc='upper left')
66    pl.show()
67

69if __name__ == '__main__':
70    main()
