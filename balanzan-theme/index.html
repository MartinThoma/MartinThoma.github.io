I don't understand why all desktop environment seem to lack a good theme chooser/editor and good themes. I have one favorite theme - called "Balanzan Theme" which should be available in every distribution. The Balanzan Theme is part of the bisigi-project. According to the package, it is licensed under GPL. So I think I can upload some of its content here.

Installation

sudo add-apt-repository ppa:bisigi
sudo apt-get update
sudo apt-get install bisigi-themes
