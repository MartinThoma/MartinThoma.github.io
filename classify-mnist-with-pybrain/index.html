Classify MNIST with PyBrain

June 18th, 2014

The MNIST database is a huge database of handwritten digits that is commonly used for training, evaluating and comparing classifiers.

It has a training set of 60,000 instances and a test set of 10,000 instances. Every instance is a 28 Ã— 28 pixel grayscale image.

The MNIST format

MNIST comes in 4 files (download here):

  • train-images-idx3-ubyte.gz: training set images (9.45 MB, 60000 instances)
  • train-labels-idx1-ubyte.gz: training set labels (28.2 kB, 60000 labels)
  • t10k-images-idx3-ubyte.gz: test set images (1.57 MB, 10000 instances)
  • t10k-labels-idx1-ubyte.gz: test set labels (4.43 kB, 10000 labels)

Both label files are like this:

[offset] [type]          [value]          [description] 
0000     32 bit integer  0x00000801(2049) magic number (MSB first) 
0004     32 bit integer  60000            number of items 
0008     unsigned byte   ??               label 
0009     unsigned byte   ??               label 
........ 
xxxx     unsigned byte   ??               label
The labels values are 0 to 9.

and both image containers are like that:

[offset] [type]          [value]          [description] 
0000     32 bit integer  0x00000803(2051) magic number 
0004     32 bit integer  60000            number of images 
0008     32 bit integer  28               number of rows 
0012     32 bit integer  28               number of columns 
0016     unsigned byte   ??               pixel 
0017     unsigned byte   ??               pixel 
........ 
xxxx     unsigned byte   ??               pixel

And they are, of course, compressed.

Reading the dataset

Python brings all necessary tools to make it easy to read the dataset:

  • gzip: A library for reading gzipped files
  • unpack to read the packed binary data

As the training and the testing dataset is structured the same way, we can create a method thad retrives the data for both files.

 1from struct import unpack
 2import gzip
 3from numpy import zeros, uint8
 4
 5
 6def get_labeled_data(imagefile, labelfile):
 7    """Read input-vector (image) and target class (label, 0-9) and return
 8       it as list of tuples.
 9    """
10    # Open the images with gzip in read binary mode
11    images = gzip.open(imagefile, 'rb')
12    labels = gzip.open(labelfile, 'rb')
13
14    # Read the binary data
15
16    # We have to get big endian unsigned int. So we need '>I'
17
18    # Get metadata for images
19    images.read(4)  # skip the magic_number
20    number_of_images = images.read(4)
21    number_of_images = unpack('>I', number_of_images)[0]
22    rows = images.read(4)
23    rows = unpack('>I', rows)[0]
24    cols = images.read(4)
25    cols = unpack('>I', cols)[0]
26
27    # Get metadata for labels
28    labels.read(4)  # skip the magic_number
29    N = labels.read(4)
30    N = unpack('>I', N)[0]
31
32    if number_of_images != N:
33        raise Exception('number of labels did not match the number of images')
34
35    # Get the data
36    x = zeros((N, rows, cols), dtype=uint8)  # Initialize numpy array
37    y = zeros((N, 1), dtype=uint8)  # Initialize numpy array
38    for i in range(N):
39        if i % 1000 == 0:
40            print("i: %i" % i)
41        for row in range(rows):
42            for col in range(cols):
43                tmp_pixel = images.read(1)  # Just a single byte
44                tmp_pixel = unpack('>B', tmp_pixel)[0]
45                x[i][row][col] = tmp_pixel
46        tmp_label = labels.read(1)
47        y[i] = unpack('>B', tmp_label)[0]
48    return (x, y)

Classify data

Now we can use PyBrain to classify data.

The following code will first build the PyBrain datastructure for the training set and the testing set. Then it will build a very simple neural network called a Multilayer Perceptron (MLP) with three layers: An input layer, a hidden layer and an output layer.

After creating it, the MLP will be trained with the backpropagation algorithm. Every training step is followed by an evaluation step.

By the way, numpy.ravel simply flattens a list.

 1from numpy import ravel
 2
 3from pybrain.datasets import ClassificationDataSet
 4from pybrain.utilities import percentError
 5from pybrain.tools.shortcuts import buildNetwork
 6from pybrain.supervised.trainers import BackpropTrainer
 7from pybrain.structure.modules import SoftmaxLayer
 8
 9
10def classify(training, testing, HIDDEN_NEURONS, MOMENTUM, WEIGHTDECAY,
11             LEARNING_RATE, LEARNING_RATE_DECAY, EPOCHS):
12    trndata = ClassificationDataSet(INPUT_FEATURES, 1, nb_classes=CLASSES)
13    tstdata = ClassificationDataSet(INPUT_FEATURES, 1, nb_classes=CLASSES)
14
15    for i in range(len(testing)):
16        tstdata.addSample(ravel(testing['x'][i]), [testing['y'][i]])
17    for i in range(len(trndata)):
18        trndata.addSample(ravel(trndata['x'][i]), [trndata['y'][i]])
19
20    # This is necessary, but I don't know why
21    # See http://stackoverflow.com/q/8154674/562769
22    trndata._convertToOneOfMany()
23    tstdata._convertToOneOfMany()
24
25    fnn = buildNetwork(trndata.indim, HIDDEN_NEURONS, trndata.outdim,
26                       outclass=SoftmaxLayer)
27
28    trainer = BackpropTrainer(fnn, dataset=trndata, momentum=MOMENTUM,
29                              verbose=True, weightdecay=WEIGHTDECAY,
30                              learningrate=LEARNING_RATE,
31                              lrdecay=LEARNING_RATE_DECAY)
32    for i in range(EPOCHS):
33        trainer.trainEpochs(1)
34        trnresult = percentError(trainer.testOnClassData(),
35                                 trndata['class'])
36        tstresult = percentError(trainer.testOnClassData(
37                                 dataset=tstdata), tstdata['class'])
38
39        print("epoch: %4d" % trainer.totalepochs,
40              "  train error: %5.2f%%" % trnresult,
41              "  test error: %5.2f%%" % tstresult)
42    return fnn
