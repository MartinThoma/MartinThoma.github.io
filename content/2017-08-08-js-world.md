## Framework vs Library

<dl>
    <dt>Framework</dt>
    <dd>You fill in the gaps. The framework calls you.</dd>
    <dt>Library</dt>
    <dd>Collection of functions. You call the library functions; you write the main thing.</dd>
</dl>
