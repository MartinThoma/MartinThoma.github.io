Create LaTeX timetable

October 18th, 2011

LaTeX is a quite cool document markup language and document preparation system. You can easily create mathematical formulas.

Today Iâ€™ve created a LaTeX timetable. Well, to be honest I have only used the timetable package from Pascal Gwosdek found on planetk.de.

Here is the LaTeX-Code:

 1\documentclass[a4paper,10pt]{report}
 2
 3% Definitions
 4\usepackage{lscape}
 5\usepackage[height=25cm]{geometry}
 6\usepackage{timetable}
 7
 8\begin{document}
 9\thispagestyle{empty}
10\begin{landscape}
11\noindent\printheading{Stundenplan von Martin Thoma - 1. Semester}
12
13% Define the layout of your time tables
14\setslotsize{2.8cm}{0.3cm}
15\setslotcount {5} {40}
16\settopheight{3}
17\settextframe{0.8mm}
18
19% Retro
20\setframetype[t]{1}
21\seteventcornerradius{0pt}
22
23% Print timestamps into event blocks
24%\setprinttimestamps{2}
25
26% Define event types
27\defineevent{lecture}{0.0} {0.28}{1.0} {1.0}{1.0}{1.0}
28\defineevent{exercise-course}    {1.0} {0.4} {0.2} {1.0}{1.0}{1.0}
29\defineevent{tutorial}   {0.6} {0.8} {1.0} {1.0}{1.0}{1.0}
30\defineevent{langcourse} {1.0} {0.4} {0.2} {1.0}{1.0}{1.0}
31\defineevent{work}       {0.21}{0.5} {0.16}{1.0}{1.0}{1.0}
32
33% Start the time table
34\begin{timetable}
35  \hours{8}{15}{1}
36  \germandays{1}
37  \event 1 {0945} {1115} {Betriebssysteme}                        {Bellosa}          {10.23 Nusselt}     {lecture}
38  \event 1 {1130} {1300} {Wahrscheinlichkeits-theorie}            {Hug}              {11.40 Tulla HS}    {lecture}
39  \event 1 {1400} {1530} {Programmieren}                          {Pretschner}       {50.35 HS a. F.}    {lecture}
40  \event 1 {1545} {1715} {LinAlg und Ana}                         {Leuzinger}        {30.46 Neue Chemie} {exercise-course}
41  \event 2 {0800} {0930} {Analysis I}                             {Schmoeger}        {30.46 Neue Chemie} {lecture}
42  \event 2 {0945} {1115} {Betriebssysteme}                        {Bellosa}          {20.40 HS 37}       {exercise-course}
43  \event 2 {1130} {1300} {Theoretische Grundlagen der Informatik} {Wagner}           {30.21 Gerthsen}    {lecture}
44  \event 3 {0800} {0930} {LinAlg und Analytische Geometrie I}     {Leuzinger}        {10.21 Daimler}     {lecture}
45  \event 3 {1400} {1530} {Grundbegriffe der Informatik}           {Schultz}          {50.35 HS a. F.}    {lecture}
46  \event 4 {0800} {0930} {Analysis I}                             {Schmoeger}        {30.46 Neue Chemie} {lecture}
47  \event 4 {1130} {1300} {Theoretische Grundlagen der Informatik} {Wagner}           {30.21 Gerthsen}    {lecture}
48  \event 5 {0800} {0930} {LinAlg und Analytische Geometrie I}     {Leuzinger}        {11.40 Tulla HS}    {lecture}
49  \event 5 {0945} {1115} {Grundbegriffe der Informatik}           {Schultz}          {50.35 HS a. F.}    {exercise-course}
50  \event 5 {1545} {1715} {Analysis I}                             {Schmoeger}        {10.21 Benz}        {exercise-course}
51\end{timetable}
52\end{landscape}
53\end{document}

Here is the timetable and the example timtable in LaTeX. If you have a Linux machine, you can create the timetable with this command: bash pdflatex example.tex -output-format=pdf
