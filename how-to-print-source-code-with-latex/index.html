How to print Source Code with LaTeX

April 29th, 2012

I often need to print source code. Some years ago for a German competition called “Bundeswettbewerb Informatik”, now for projects at my university. If you use LaTeX, you can simply include the source code into your document! Here are three examples with listings and minted. I’ve also included example PDF files.

Minimal example

LaTeX Java Source Code: listings

Here is an minimal example how you could print Source Code with LaTeX: 

 1\documentclass[a4paper,12pt]{article}
 2\usepackage{amssymb} % needed for math
 3\usepackage{amsmath} % needed for math
 4\usepackage[utf8]{inputenc} % this is needed for german umlauts
 5\usepackage[ngerman]{babel} % this is needed for german umlauts
 6% this is needed for correct output of umlauts in pdf
 7\usepackage[T1]{fontenc}    
 8\usepackage[margin=2.5cm]{geometry} %layout
 9\usepackage{listings} % needed for the inclusion of source code
10
11% this is needed for forms and links within the text
12\usepackage{hyperref}  
13
14%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
15% THE DOCUMENT BEGINS                      %
16%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
17\begin{document}
18        \lstinputlisting[language=Java]{Othello.java}
19\end{document}

My Template

If you want to customize a little bit more and if you want to get highlighted (colorized) source code, you could use the following template. It looks like this as a PDF-file.

 2\usepackage{amssymb} % needed for math

 4\usepackage[utf8]{inputenc} % this is needed for german umlauts
 5\usepackage[ngerman]{babel} % this is needed for german umlauts
 6\usepackage[T1]{fontenc}    % this is needed for correct output of umlauts in pdf
 7\usepackage[margin=2.5cm]{geometry} %layout
 8\usepackage{listings} % needed for the inclusion of source code
 9
10% the following is needed for syntax highlighting
11\usepackage{color}
12 
13\definecolor{dkgreen}{rgb}{0,0.6,0}
14\definecolor{gray}{rgb}{0.5,0.5,0.5}
15\definecolor{mauve}{rgb}{0.58,0,0.82}
16
17\lstset{ %
18  language=Java,                  % the language of the code
19  basicstyle=\footnotesize,       % the size of the fonts that are used for the code
20  numbers=left,                   % where to put the line-numbers
21  numberstyle=\tiny\color{gray},  % the style that is used for the line-numbers
22  stepnumber=1,                   % the step between two line-numbers. If it's 1, each line 
23                                  % will be numbered
24  numbersep=5pt,                  % how far the line-numbers are from the code
25  backgroundcolor=\color{white},  % choose the background color. You must add \usepackage{color}
26  showspaces=false,               % show spaces adding particular underscores
27  showstringspaces=false,         % underline spaces within strings
28  showtabs=false,                 % show tabs within strings adding particular underscores
29  frame=single,                   % adds a frame around the code
30  rulecolor=\color{black},        % if not set, the frame-color may be changed on line-breaks within not-black text (e.g. commens (green here))
31  tabsize=4,                      % sets default tabsize to 2 spaces
32  captionpos=b,                   % sets the caption-position to bottom
33  breaklines=true,                % sets automatic line breaking
34  breakatwhitespace=false,        % sets if automatic breaks should only happen at whitespace
35  title=\lstname,                 % show the filename of files included with \lstinputlisting;
36                                  % also try caption instead of title
37  keywordstyle=\color{blue},          % keyword style
38  commentstyle=\color{dkgreen},       % comment style
39  stringstyle=\color{mauve},         % string literal style
40  escapeinside={\%*}{*)},            % if you want to add a comment within your code
41  morekeywords={*,...}               % if you want to add more keywords to the set
42}
43
44% this is needed for forms and links within the text
45\usepackage{hyperref}  
46
47%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
48% Variablen                                                                                  %
49%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%

52\title{This is the title}
53\author{\authorName}
54\date{\today}
55
56%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
57% PDF Meta information                                                                  %
58%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
59\hypersetup{
60  pdfauthor   = {\authorName},
61  pdfkeywords = {\tags},
62  pdftitle    = {This is the title}
63} 
64
65%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
66% THE DOCUMENT BEGINS                                                            %
67%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
68\begin{document}
69        \lstinputlisting[language=Java]{Othello.java}
70\end{document}

Example

LaTeX Java Source Code: minted

This is the PDF-file produced by the following LaTeX-Code:

 1\documentclass[a4paper,12pt]{article}
 2\usepackage{amssymb} % needed for math
 3\usepackage{amsmath} % needed for math
 4\usepackage[utf8]{inputenc} % this is needed for german umlauts
 5\usepackage[ngerman]{babel} % this is needed for german umlauts
 6\usepackage[T1]{fontenc}    % this is needed for correct output of umlauts in pdf
 7\usepackage[margin=2cm]{geometry} %layout
 8\usepackage{minted} % needed for the inclusion of source code
 9
10\begin{document}
11\renewcommand{\theFancyVerbLine}{
12  \sffamily\textcolor[rgb]{0.5,0.5,0.5}{\scriptsize\arabic{FancyVerbLine}}}
13
14\inputminted[linenos, numbersep=5pt, tabsize=4, frame=lines, label=Othello.java]{java}{Othello.java}
15
16\end{document}
