Predictions for 2019

Machine Learning and Technology

  1. Tensorflow will keep being the dominant deep learning framework as measured by (1) StackOverflow questions, (2) Google Trend, (3) Github repositories
  2. There will still not be a DOTA playing AI which can replace a human DOTA player in a human team with the usual flexibility.
  3. We will see fraud cases related to ML (e.g. using speech synthesis for identity theft or tricking a ML system)
