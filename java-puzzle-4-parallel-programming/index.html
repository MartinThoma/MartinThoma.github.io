Explanation

If you start threads like this, you don’t get any guarantee that they will finish their execution in order. If you want them to execute in block of four, you could use join():

 1public class test {
 2    public static void main(String[] args) {
 3        myParallelTry a = new myParallelTry("A");
 4        myParallelTry b = new myParallelTry("B");
 5        myParallelTry c = new myParallelTry("C");
 6        myParallelTry d = new myParallelTry("D");
 7
 8        Thread tA = new Thread(a); tA.start();
 9        Thread tB = new Thread(b); tB.start();
10        Thread tC = new Thread(c); tC.start();
11        Thread tD = new Thread(d); tD.start();
12        try {
13            tA.join();
14            tB.join();
